## About

GitHub is the working home for my long-horizon projects across applied AI, machine learning, agent systems, and technical education. Most repositories are built as learning tracks: practical, inspectable, and designed to turn ideas into working code.

Areas I focus on:

- applied AI systems, ML pipelines, and agent workflows
- repo-first education designed for hands-on learning
- multilingual publishing across English, Chinese, and Italian
- long-term projects spanning data, finance, tooling, and creator infrastructure
